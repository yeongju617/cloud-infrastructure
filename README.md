# Cloud Infrastructure

## 📌 Project Overview

실제 서비스를 운영한다고 가정하고 AWS 기반의 클라우드 인프라를 설계하고 구축하는 프로젝트입니다.

목표는 단순히 서버를 만드는 것이 아니라,
- 네트워크 설계
- 배포 자동화(CI/CD)
- 모니터링
- 로그 관리
- 장애 대응

까지 경험하는 것입니다.
